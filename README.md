# 🏧 ATM Simulator in Python

A simple Python based ATM Simulator with PIN authentication.

## Features
- PIN verification with 3 attempts
- Check Balance
- Deposit Money
- Withdraw Money  
- Mini Statement

## How to Run
1. Clone this repo
2. Run: python "ATM Simulator.py.py"
3. Enter PIN: 1234
